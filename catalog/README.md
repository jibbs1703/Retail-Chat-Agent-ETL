# Fashion Catalog Web Scraper

This directory contains a high-performance, asynchronous web scraper designed to extract product metadata 
and image assets from Fashion Nova by leveraging Python's `asyncio` and `aiohttp` libraries. The script 
uses a non-blocking architecture to gather data from collection pages and individual product detail pages
simultaneously.

The scraper extracts the following information:
- Product Title and Price
- Image URLs (filtered for uniqueness)
- Product Details and Material Descriptions
- Financing and Installment Metadata
- Size Availability
- Promotional Taglines
